Movie API Documentation

Base URL

http://localhost:8080

Endpoints

Method Endpoint Description
GET /movies Returns a list of all movies.
GET /movies/:Title Returns details of a movie by its title.
GET /genres/:Name Returns the genre details of a movie by name.
GET /directors/:Name Returns the director's information based on their name.
GET /actors/:Name/movies Returns movies that feature the specified actor.
GET /movies/:Title/rating Returns the rating of the specified movie.
GET /movies/:Title/releaseyear Returns the release year of the specified movie.
GET /users Returns a list of all users.
POST /users Creates a new user.
PUT /users/:Username Updates the user details.
POST /users/:Username/movies/:MovieID Adds a movie to the user's favorite list.
GET /users/:Username/movies Returns the user's favorite movies.
DELETE /users/:Username/movies/:MovieID Removes a movie from the user's favorite list.
DELETE /users/:Username Deletes a user.

Request and Response Examples

GET /genres/:Name

Returns the genre details of a movie by name.

Response Example:

  {
    "Name": "Action",
    "Description": "Action movies feature fast-paced sequences, physical stunts, and high-intensity scenes."
  }

GET /directors/:Name

Returns the director details by name.

Response Example:

  {
    "_id": "603d0e21718e7315b8e3f8b1",
    "Name": "Christopher Nolan",
    "Bio": "Christopher Nolan is a renowned director known for his work in drama and action movies."
  }

Get Movie by Title (GET /movies/:Title)

Response Example:

{
  "_id": "603d0b43718e7315b8e3f8a2",
  "Title": "Movie Title",
  "Description": "Movie Description",
  "Genre": {
    "Name": "Genre Name",
    "Description": "Genre Description"
  },
  "Director": {
    "Name": "Director Name",
    "Bio": "Director Bio",
    "Birth": "1960-01-01"
  },
  "Actors": ["Actor 1", "Actor 2"],
  "ReleaseYear": "1999",
  "Rating": 8.5
}

User Registration (POST /users)

Request Body Example:

{
  "Username": "username",
  "Password": "password",
  "Email": "user@example.com",
  "Birth_date": "1990-01-01"
}

Response Example:

{
  "_id": "603d0b43718e7315b8e3f8a2",
  "Username": "username",
  "Email": "user@example.com",
  "Birth_date": "1990-01-01",
  "FavoriteMovies": []
}
